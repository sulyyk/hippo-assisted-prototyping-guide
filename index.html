AI-assisted prototyping with Gemini CLI

Learn how to use Gemini CLI alongside the GOV.UK Prototype Kit to build realistic prototypes faster – even if you've never written a line of code.

This guide is for designers and other professions who want to use AI tools to speed up prototyping. You do not need any coding experience.

What you'll learn

By the end of this guide, you'll have built a working GOV.UK prototype for a fictional 'Report a building issue' service – using plain English prompts instead of writing code by hand.

The guide has 2 parts:

Part 1: Get set up

Install everything you need and check it all works. This takes about 30 minutes.

  2. Install a code editor
  3. Install Node.js
  4. Install the GOV.UK Prototype Kit
  5. Install Gemini CLI
