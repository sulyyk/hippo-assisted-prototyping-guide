Part 1: Get set up

Install a code editor

A code editor is where you will see the files in your prototype. You will also use its built-in terminal to run Gemini CLI.

We recommend Visual Studio Code (VS Code). It is free and works on Mac, Windows, and Linux.

Step 1: Download VS Code

Go to code.visualstudio.com and download the version for your computer.

Step 2: Install it

Open the downloaded file and follow the installation instructions. Accept the default settings – you do not need to change anything.

Step 3: Open VS Code

Once installed, open VS Code. You should see a welcome screen.

Checkpoint: You should now have VS Code open on your screen. If you see a welcome tab or an empty editor window, you are ready to move on.

Step 4: Open the terminal in VS Code

You will use the terminal a lot in this guide, so it is worth learning how to open it now.

In VS Code, press:

  • Mac: Ctrl + ` (the backtick key, usually above the tab key)
  • Windows: Ctrl + ' (the apostrophe key)

Or go to the menu: Terminal → New Terminal.

A panel should appear at the bottom of the VS Code window. This is your terminal.

Checkpoint: You should see a terminal panel at the bottom of VS Code with a blinking cursor. It might show your username and computer name. This is normal.

The terminal looks intimidating

Yes. If you have never used a terminal before, it can feel strange to type commands instead of clicking buttons. That is completely normal.

You do not need to learn terminal commands. This guide will tell you exactly what to type each time. Think of it like following a recipe – you do not need to understand the chemistry of baking to follow the steps.
